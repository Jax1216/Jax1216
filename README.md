## Featured Projects
 
### ⬡ xVector — Tactical Audio Intelligence
> Real-time audio classification and transcription system built for field deployment.
 
ESP32 I2S microphone firmware streams audio over UDP to a Flask backend, where **YAMNet/TensorFlow** classifies sound events in parallel with **Whisper** transcription. A React dashboard surfaces alerts, a live Leaflet.js sensor map, and a Commander Guidance Worksheet PDF generator with Ollama chat integration.
 
`ESP32` `Flask` `TensorFlow` `Whisper` `React` `Leaflet.js` `UDP` `Python`
